api: 'DELETE /v1/voice/campaigns/{campaign_id}'
description: 'Delete a voice campaign'
---

## Description

Deletes a voice campaign and its associated contacts. This action cannot be undone.

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Your Teli API key
</ParamField>

## Path Parameters

<ParamField path="campaign_id" type="string" required>
  Campaign unique identifier to delete
  
  Example: `"vc 1"`
</ParamField>

## Response Fields

<ResponseField name="success" type="boolean">
  Whether the deletion was successful
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message
</ResponseField>

<ResponseField name="powered_by" type="string">
  Always returns "Teli"
</ResponseField>

## Example Request

```bash cURL
curl -X DELETE "https://api.teli.ai/v1/voice/campaigns/vc%201" \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.teli.ai/v1/voice/campaigns/vc 1', {
  method: 'DELETE',
  headers: {
    'X-API-Key': 'YOUR_API_KEY'
  }
});
```

## Example Response

```json 200
{
  "success": true,
  "message": "Campaign vc 1 deleted",
  "powered_by": "Teli"
}
```

```json 404
{
  "success": false,
  "error": "Campaign not found",
  "powered_by": "Teli"
}
```

## Notes

- Deletes campaign from DynamoDB (teli_voice_campaigns table)
- Deletes all associated contacts from teli_voice_contacts table
- Does NOT delete call history or recordings (those are preserved)
- Requires exact campaign_id match
- URL encode the campaign_id if it contains spaces (e.g., "vc 1" â†’ "vc%201")
- Cannot be undone - use with caution
