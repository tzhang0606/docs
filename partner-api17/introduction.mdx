## Base URL

```
https://api.teli.ai
```

<Note>
  Replace `https://api.teli.ai` with your actual API URL provided during onboarding. You can edit the URL directly in the "Try it" playground on each endpoint page.
</Note>

## Authentication

All requests require an API key:

```bash
X-API-Key: YOUR_API_KEY
```

## Key Features

- **Organizations** - Multi-tenant architecture with organization and user management
- **AI Agents** - Intelligent SMS and voice conversation agents with custom prompts
- **Voice Agent Tools** - 10 tool types including end call, transfer, custom API, Cal.com booking, agent transfer, send SMS, and more
- **Knowledge Bases** - Enhance voice agents with URLs, documents, and custom knowledge for accurate responses
- **SMS Campaigns** - Automated SMS outreach with drip sequences and AI-powered follow-ups
- **Voice Campaigns** - Bulk voice calling with intelligent conversation handling
- **Call History** - Complete call analytics, recordings, and transcripts with S3 storage
- **Phone Numbers** - Voice phone number provisioning and management
- **Contacts & Leads** - Contact list management and lead tracking
- **10DLC Registration** - Brand and campaign registration for compliant SMS messaging (US)
- **Analytics** - Comprehensive usage and cost tracking with detailed reporting
- **Multi-Tenant Isolation** - Secure data separation per organization and tenant
- **Scalable Infrastructure** - Built for high-volume messaging and calling with auto-scaling
