---
title: 'Create 10DLC Campaign'
api: 'POST /v1/10dlc/campaign'
description: 'Submit campaign for carrier approval'
---

Create a 10DLC campaign registration with sample messages and compliance details.

<Warning>
Campaign must include at least 3 sample messages and clear opt-in/opt-out messaging.
</Warning>

## Authorizations

<ParamField header="X-API-Key" type="string" required>
  Authentication header
</ParamField>

## Body

<ParamField body="brandId" type="string" required>
  Brand ID from brand registration
</ParamField>

<ParamField body="usecase" type="string" required>
  Campaign use case: `CUSTOMER_CARE`, `2FA`, `MARKETING`, `ACCOUNT_NOTIFICATION`
</ParamField>

<ParamField body="description" type="string" required>
  Campaign description (40-4096 characters)
</ParamField>

<ParamField body="messageFlow" type="string" required>
  How subscribers opt-in (40-4096 characters)
</ParamField>

<ParamField body="sample1" type="string" required>
  Sample message 1 (20-1024 characters). Must include opt-out language.
</ParamField>

<ParamField body="sample2" type="string" required>
  Sample message 2
</ParamField>

<ParamField body="sample3" type="string" required>
  Sample message 3
</ParamField>

<ParamField body="sample4" type="string">
  Sample message 4 (optional but recommended)
</ParamField>

<ParamField body="sample5" type="string">
  Sample message 5 (optional but recommended)
</ParamField>

<ParamField body="directLending" type="boolean">
  Is this for direct lending? Default: `false`
</ParamField>

<ParamField body="ageGated" type="boolean">
  Age-restricted content? Default: `false`
</ParamField>

<ParamField body="subscriberOptin" type="boolean">
  Requires opt-in? Default: `true`
</ParamField>

<ParamField body="subscriberOptout" type="boolean">
  Has opt-out? Default: `true`
</ParamField>

<ParamField body="subscriberHelp" type="boolean">
  Has help keywords? Default: `true`
</ParamField>

<ParamField body="optinKeywords" type="string">
  Opt-in keywords. Default: `"START, YES, SUBSCRIBE"`
</ParamField>

<ParamField body="optoutKeywords" type="string">
  Opt-out keywords. Default: `"STOP, QUIT, UNSUBSCRIBE, END"`
</ParamField>

<ParamField body="helpKeywords" type="string">
  Help keywords. Default: `"HELP, INFO"`
</ParamField>

<ParamField body="helpMessage" type="string">
  Message sent when user requests help (20-320 characters)
</ParamField>

<ParamField body="optoutMessage" type="string">
  Message sent when user opts out
</ParamField>

## Response

<ResponseField name="campaignId" type="string">
  Campaign registration ID
</ResponseField>

<ResponseField name="mnoMetadata" type="object">
  Carrier approval status for each carrier (AT&T, T-Mobile, Verizon, etc.)
</ResponseField>

## Next Steps

After creating a campaign:
1. Save the `campaignId` to your organization
2. Wait for carrier approval (1-2 days)
3. Webhook will notify when approved
4. Phone number assigned automatically
