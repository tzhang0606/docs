---
title: 'Get SubGroup Details'
api: 'GET /v1/10dlc/subgroup/{subgroup_id}'
description: 'Get detailed information about a specific subgroup'
---

Retrieve full details for a specific subgroup, including its name, description, and status.

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Your API key
</ParamField>

## Path Parameters

<ParamField path="subgroup_id" type="string" required>
  SubGroup ID (e.g., `SQ8YFT`)
</ParamField>

## Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.teli.ai/v1/10dlc/subgroup/SQ8YFT" \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.teli.ai/v1/10dlc/subgroup/SQ8YFT',
  {
    headers: { 'X-API-Key': 'YOUR_API_KEY' }
  }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.teli.ai/v1/10dlc/subgroup/SQ8YFT',
    headers={'X-API-Key': 'YOUR_API_KEY'}
)
data = response.json()
```
</CodeGroup>

## SubGroup Fields

| Field | Type | Description |
|-------|------|-------------|
| `sub_group_id` | string | Unique subgroup identifier |
| `sub_group_name` | string | Display name of the subgroup |
| `sub_group_description` | string | Description of the subgroup |
| `group_id` | string | Parent group identifier |
| `status` | string | Current status (`Active`, `Inactive`) |
