---
title: 'List Organizations'
api: 'GET /v1/organizations'
description: 'List all organizations'
---

Retrieve all organizations accessible to your API key.

## Authorizations

<ParamField header="X-API-Key" type="string" required>
  Authentication header containing your API key.
</ParamField>

<RequestExample>

```bash cURL
curl -X GET https://api.example.com/v1/organizations \
  -H "X-API-Key: YOUR_API_KEY"
```

</RequestExample>
