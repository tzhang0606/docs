---
title: 'Get Usage Metrics'
api: 'GET /v1/analytics/usage'
description: 'Retrieve SMS/MMS segment counts and voice call seconds for an organization or user'
---

## Endpoint

```
GET /v1/analytics/usage
```

## Authentication

Requires `X-API-Key` header with your API key.

## Query Parameters

<ParamField query="organization_id" type="string">
  Organization unique ID to get usage for. Either `organization_id` or `user_id` is required.
</ParamField>

<ParamField query="user_id" type="string">
  User unique ID to get usage for. Either `organization_id` or `user_id` is required.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="usage" type="object">
  Usage metrics object
  
  <Expandable title="usage properties">
    <ResponseField name="organization_id" type="string">
      Organization unique ID (when querying by org)
    </ResponseField>
    
    <ResponseField name="organization_name" type="string">
      Organization name (when querying by org)
    </ResponseField>
    
    <ResponseField name="user_id" type="string">
      User unique ID (when querying by user)
    </ResponseField>
    
    <ResponseField name="user_name" type="string">
      User name (when querying by user)
    </ResponseField>
    
    <ResponseField name="user_email" type="string">
      User email (when querying by user)
    </ResponseField>
    
    <ResponseField name="sms" type="object">
      SMS segment counts
      
      <Expandable title="sms properties">
        <ResponseField name="outbound" type="integer">
          Total outbound SMS segments sent
        </ResponseField>
        
        <ResponseField name="inbound" type="integer">
          Total inbound SMS segments received
        </ResponseField>
        
        <ResponseField name="total" type="integer">
          Total SMS segments (outbound + inbound)
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="mms" type="object">
      MMS segment counts
      
      <Expandable title="mms properties">
        <ResponseField name="outbound" type="integer">
          Total outbound MMS segments sent
        </ResponseField>
        
        <ResponseField name="inbound" type="integer">
          Total inbound MMS segments received
        </ResponseField>
        
        <ResponseField name="total" type="integer">
          Total MMS segments (outbound + inbound)
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="voice" type="object">
      Voice call seconds
      
      <Expandable title="voice properties">
        <ResponseField name="seconds" type="integer">
          Total regular voice call seconds
        </ResponseField>
        
        <ResponseField name="kb_seconds" type="integer">
          Total knowledge base-enhanced voice call seconds
        </ResponseField>
        
        <ResponseField name="total_seconds" type="integer">
          Total voice seconds (seconds + kb_seconds)
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="period" type="string">
      Time period for metrics (currently always "all_time")
    </ResponseField>
  </Expandable>
</ResponseField>

## Notes

- All counters represent **all-time totals** since account creation
- SMS/MMS counts are in **segments** (not messages) — a long message may count as multiple segments
- Voice seconds include both regular calls and knowledge base-enhanced calls
- `kb_seconds` tracks calls that used knowledge base retrieval (premium feature)
- Counters are updated in real-time as usage occurs
