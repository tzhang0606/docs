---
title: 'Remove from DNC'
api: 'POST /v1/dnc/remove'
description: 'Remove phone from Do Not Call list'
---

## Authorizations

<ParamField header="X-API-Key" type="string" required>
  Authentication header
</ParamField>

## Body

<ParamField body="phone_number" type="string" required>
  Phone number in E.164 format
</ParamField>

<ParamField body="organization_id" type="string" required>
  Organization unique_id
</ParamField>

<RequestExample>

```bash cURL
curl -X POST https://api.example.com/v1/dnc/remove \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "phone_number": "+15551234567",
    "organization_id": "{org_id}"
  }'
```

</RequestExample>

<ResponseExample>

```json 200
{
  "success": true,
  "message": "Phone number removed from DNC list",
  "phone_number": "+15551234567",
  "organization_id": "{org_id}",
  "powered_by": "Teli"
}
```

</ResponseExample>
