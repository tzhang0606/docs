## What is the Teli MCP Server?

The Teli MCP Server connects Claude to the Teli AI platform. Create and manage AI-powered SMS and voice agents, launch outbound campaigns, organize contacts, and monitor campaign progress — all through natural conversation with Claude.

No dashboard switching. Just tell Claude what you want to do.

<CardGroup cols={2}>
  <Card title="Agent Management" icon="robot">
    Create and configure SMS and voice AI agents with custom prompts, extraction fields, and drip schedules.
  </Card>
  <Card title="SMS Campaigns" icon="message-sms">
    Launch outbound AI-driven SMS campaigns with contact lists, MMS support, and optional voice escalation.
  </Card>
  <Card title="Voice Campaigns" icon="phone">
    Run bulk outbound AI voice call campaigns with scheduling, number pools, and redial logic.
  </Card>
  <Card title="Contact Management" icon="users">
    Create contact groups, upload bulk lists, and add individual contacts with rich profile fields.
  </Card>
  <Card title="Campaign Analytics" icon="chart-bar">
    Check campaign progress, completion rates, and voice call outcomes.
  </Card>
  <Card title="Phone Numbers" icon="phone-plus">
    Provision new voice phone numbers by area code directly from Claude.
  </Card>
</CardGroup>

## Setup

<Note>Teli is currently pending review for listing in the Anthropic Connectors Directory. In the meantime, you can connect manually by adding the MCP server URL as a custom connector in Claude.</Note>

<Warning>The Teli MCP Server is available to users on the **Scale or Enterprise plan** only. If you are on the Growth plan, please [contact us](mailto:info@teli.ai) to discuss upgrading your account.</Warning>

<Steps>
  <Step title="Open Claude and go to Connectors">
    Visit [claude.com/connectors](https://claude.com/connectors) or open Connectors in your Claude settings.
  </Step>
  <Step title="Add a custom connector">
    Click **Add custom connector** and enter the following MCP server URL:
    ```
    https://teli--mcp-server-fastapi-app.modal.run/mcp
    ```
  </Step>
  <Step title="Authenticate">
    Complete OAuth authentication with your Teli account credentials.
  </Step>
  <Step title="Start using it">
    Ask Claude to manage your agents, campaigns, or contacts.
  </Step>
</Steps>
