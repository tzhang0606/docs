## Description

Lists all calls from your organization's call history (stored in Teli Voice database). Returns call metadata including duration, sentiment, and recording URLs.

**Data Source**: Teli Voice call history database (NOT live provider data)

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Your Teli API key
</ParamField>

## Query Parameters

<ParamField query="organization_id" type="string" required>
  Organization unique_id to filter calls
</ParamField>

<ParamField query="user_id" type="string">
  User unique_id (required for non-admin users to filter their calls)
</ParamField>

<ParamField query="is_admin" type="boolean" default="false">
  If true, returns all organization calls. If false, returns only user's calls.
</ParamField>

<ParamField query="agent_id" type="string">
  Filter by specific agent ID (optional)
</ParamField>

<ParamField query="call_status" type="string" default="ended">
  Filter by call status: `ended`, `ongoing`, `error`
</ParamField>

<ParamField query="direction" type="string">
  Filter by direction: `inbound` or `outbound`
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Number of results (max: 100)
</ParamField>

## Notes

- Calls are stored in Teli Voice database after completion
- Recordings are hosted on Teli Voice Storage (S3)
- For full call transcript, use `GET /v1/voice/calls/{call_id}`
- Admin users see all organization calls
- Regular users see only their own calls
- Results are ordered by start_timestamp (newest first)
