## Authentication

All requests require an API key:

```bash
X-API-Key: YOUR_API_KEY
```

## Key Features

- **Organizations** - Multi-tenant architecture
- **AI Agents** - Intelligent SMS and voice conversation agents
- **Campaigns** - Automated SMS and voice outreach
- **Call History** - Complete call analytics and recordings
- **Analytics** - Usage and cost tracking
- **Scalable** - Built for high-volume messaging and calling
