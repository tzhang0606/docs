---
title: 'List Knowledge Bases'
api: 'GET /v1/knowledge-bases'
description: 'List all knowledge bases for an organization'
---

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Your Teli API key
</ParamField>

---

## Query Parameters

<ParamField query="organization_id" type="string" required>
  Organization unique_id to list KBs for
</ParamField>

<ParamField query="user_id" type="string" required>
  User unique_id (for filtering if not admin)
</ParamField>

<ParamField query="is_admin" type="boolean">
  If true, returns all KBs for organization. If false, returns only user's KBs. Default: false
</ParamField>

---

## Response

<ResponseField name="success" type="boolean">
  Whether request was successful
</ResponseField>

<ResponseField name="knowledge_bases" type="array">
  Array of knowledge base objects
</ResponseField>

<ResponseField name="count" type="number">
  Number of knowledge bases returned
</ResponseField>

<ResponseField name="powered_by" type="string">
  Platform identifier ("Teli")
</ResponseField>

### Knowledge Base Object Fields

<ResponseField name="knowledge_base_id" type="string">
  Unique KB identifier (use this to link to agents)
</ResponseField>

<ResponseField name="knowledge_base_name" type="string">
  KB name
</ResponseField>

<ResponseField name="status" type="string">
  Processing status: `in_progress`, `complete`, or `error`
</ResponseField>

<ResponseField name="total_sources" type="number">
  Total number of sources (URLs + texts + files)
</ResponseField>

<ResponseField name="source_types" type="array">
  Types of sources: `["url", "text", "document"]`
</ResponseField>

<ResponseField name="source_details" type="object">
  Summary of sources with URLs list and text snippet info
</ResponseField>

<ResponseField name="enable_auto_refresh" type="boolean">
  Whether auto-refresh is enabled for URLs
</ResponseField>

<ResponseField name="default_top_k" type="number">
  Default chunks to retrieve (1-10)
</ResponseField>

<ResponseField name="default_filter_score" type="number">
  Default similarity threshold (0-1)
</ResponseField>

## Use Cases

### View All KBs
```
GET /v1/knowledge-bases?organization_id=xxx&user_id=xxx&is_admin=true
→ Returns all KBs for organization
```

### View User's KBs Only
```
GET /v1/knowledge-bases?organization_id=xxx&user_id=xxx&is_admin=false
→ Returns only KBs created by this user
```
