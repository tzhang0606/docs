# 10DLC Field Reference

Complete reference for all fields, values, and options in 10DLC registration.

## Entity Types

| Value | Description |
|-------|-------------|
| `PRIVATE_PROFIT` | Private for-profit company (most common) |
| `PUBLIC_PROFIT` | Publicly traded company |
| `NON_PROFIT` | Non-profit organization |
| `GOVERNMENT` | Government entity (US only) |
| `SOLE_PROPRIETOR` | Individual/sole proprietor |

## Sample Message Requirements

- **Minimum length**: 20 characters
- **Maximum length**: 1,024 characters
- **Required count**: 3 minimum (5 recommended)
- **Must include**: Opt-out language (e.g., "Reply STOP to opt out")

## Compliance Booleans

| Field | Description | Default |
|-------|-------------|---------|
| `directLending` | Direct lending/payday loans? | `false` |
| `ageGated` | Age-restricted content? | `false` |
| `subscriberOptin` | Requires user opt-in? | `true` |
| `subscriberOptout` | Provides opt-out? | `true` |
| `subscriberHelp` | Provides help keywords? | `true` |
| `embeddedLink` | Contains URLs? | `false` |
| `embeddedPhone` | Contains phone numbers? | `false` |
| `numberPool` | Uses number pooling? | `false` |

## Default Keywords

| Type | Default Value |
|------|---------------|
| Opt-in | `"START, YES, SUBSCRIBE"` |
| Opt-out | `"STOP, QUIT, UNSUBSCRIBE, END"` |
| Help | `"HELP, INFO"` |

<Info>
Carriers require specific keywords. Using the defaults ensures compliance.
</Info>
