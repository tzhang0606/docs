---
title: 'Get Organization'
api: 'GET /v1/organizations/{id}'
description: 'Retrieve organization details'
---

Get details of a specific organization including SMS status and phone number.

## Authorizations

<ParamField header="X-API-Key" type="string" required>
  Authentication header containing your API key.
</ParamField>

## Path

<ParamField path="id" type="string" required>
  Organization unique_id
</ParamField>

<RequestExample>

```bash cURL
curl -X GET https://api.example.com/v1/organizations/{org_id} \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(`https://api.example.com/v1/organizations/${orgId}`, {
  headers: {'X-API-Key': 'YOUR_API_KEY'}
});
```

</RequestExample>

<ResponseExample>

```json 200
{
  "organization": {
    "unique_id": "1763070568417x631086913669115287",
    "name": "My Company LLC",
    "sms_active": true,
    "teli_sms_number": "+13135559999",
    "sms_registration_status": "approved",
    "total_sms_outbound": 150,
    "total_sms_inbound": 45
  },
  "powered_by": "Teli"
}
```

</ResponseExample>

## Response

<ResponseField name="unique_id" type="string">
  Organization identifier
</ResponseField>

<ResponseField name="sms_active" type="boolean">
  Whether organization can send SMS campaigns
</ResponseField>

<ResponseField name="teli_sms_number" type="string">
  Assigned 10DLC phone number
</ResponseField>

<ResponseField name="total_sms_outbound" type="number">
  Total SMS sent (all-time)
</ResponseField>
