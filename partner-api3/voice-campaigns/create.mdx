## Description

Creates and starts a bulk voice campaign using your voice agent. Calls contacts in your leads list and conducts AI-powered conversations.

**Use Cases:**
- Outbound sales calls
- Customer surveys
- Appointment reminders
- Lead qualification

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Your Teli API key
</ParamField>

## Request Body

<ParamField body="leads" type="array" required>
  Array of contact objects to call
  
  Each contact object:
  ```json
  {
    "phone_number": "+14155551234",  // Required
    "first_name": "John",            // Optional
    "last_name": "Doe",               // Optional
    "email": "john@example.com",      // Optional
    // ... any custom fields for dynamic variables
  }
  ```
</ParamField>

<ParamField body="voice_agent_id" type="string" required>
  Voice agent ID to use for calls (e.g., "agent_86f826bf8f59bacd5c10ae948a")
</ParamField>

<ParamField body="agent_outbound_number" type="string" required>
  Default outbound phone number for the campaign (E.164 format)
  
  Example: `"+15174686941"`
</ParamField>

<ParamField body="organization_id" type="string" required>
  Organization unique_id that owns this campaign
</ParamField>

<ParamField body="tenant_id" type="string" required>
  Tenant identifier for multi-tenant isolation
</ParamField>

<ParamField body="campaign_id" type="string">
  Optional campaign identifier. Auto-generated if not provided.
  
  Default: `"voice_campaign_{random}"`
</ParamField>

<ParamField body="number_pool" type="array">
  Optional pool of phone numbers to rotate for outbound calls
  
  Example:
  ```json
  [
    {"phone_number": "+14155550001"},
    {"phone_number": "+14155550002"}
  ]
  ```
</ParamField>

<ParamField body="campaign_begin_time" type="timestamp">
  When to start calling (ISO 8601 format)
  
  Default: Immediately
</ParamField>

<ParamField body="campaign_end_time" type="timestamp">
  When to stop calling (ISO 8601 format)
  
  Default: 7 days from now
</ParamField>

<ParamField body="redial_enabled" type="boolean">
  Whether to retry failed calls
  
  Default: `false`
</ParamField>

<ParamField body="redial_config" type="object">
  Retry configuration if `redial_enabled` is true
  
  ```json
  {
    "retry_limit": 3,           // Max retry attempts
    "call_interval_days": 1     // Days between retries
  }
  ```
</ParamField>

## Response Fields

<ResponseField name="success" type="boolean">
  Whether the campaign was created successfully
</ResponseField>

<ResponseField name="campaign_id" type="string">
  Unique campaign identifier
</ResponseField>

<ResponseField name="contacts_count" type="integer">
  Number of valid contacts added to campaign
</ResponseField>

<ResponseField name="invalid_leads_count" type="integer">
  Number of invalid contacts (missing phone numbers)
</ResponseField>

<ResponseField name="campaign_begin_time" type="timestamp">
  When campaign will start calling
</ResponseField>

<ResponseField name="campaign_end_time" type="timestamp">
  When campaign will stop calling
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="powered_by" type="string">
  Always returns "Teli"
</ResponseField>

## Notes

- Calls are made asynchronously after campaign creation
- Campaign status can be tracked via `GET /v1/voice/campaigns`
- All calls are recorded and stored in Teli Voice Storage (S3)
- Call history is available via `GET /v1/voice/calls`
- Voice agents must be created first via `POST /v1/agents`
- Phone numbers must be provisioned first via `POST /v1/voice/phone-numbers/create`

## Related Endpoints

- `GET /v1/voice/campaigns` - List campaigns
- `DELETE /v1/voice/campaigns/{id}` - Delete campaign
- `POST /v1/agents` - Create voice agent
- `GET /v1/voice/calls` - View call history
