## Welcome

There are two types of AI agents on Teli: Voice and SMS. Voice agents are great for handling phone calls, and SMS agents are designed for automating text-messaging.

## Authentication

All API endpoints are authenticated using Bearer tokens and picked up from the specification file.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
