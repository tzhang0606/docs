## Overview

Dynamic variables in Mintlify allow you to create personalized documentation by inserting user-specific or context-aware values into your content.

---

## Available Dynamic Variables

### **User Variables**

<ResponseField name="{{ user_name }}" type="text">
  The name of the logged-in user. **Example:** `{{ user_name }}`
</ResponseField>

<ResponseField name="{{ user_email }}" type="text">
  The email address of the logged-in user. **Example:** `{{ user_email }}`
</ResponseField>

<ResponseField name="{{ user_role }}" type="text">
  The role of the user in the system. **Example:** `{{ user_role }}`
</ResponseField>

---

### **Date & Time Variables**

<Expandable title="Date & Time Variables">
  <ResponseField name="{{ current_date }}" type="text">
    Inserts the current date. **Example:** `{{ current_date }}`
  </ResponseField>
  <ResponseField name="{{ current_time }}" type="text">
    Inserts the current time. **Example:** `{{ current_time }}`
  </ResponseField>
</Expandable>

---

### **Project Variables**

<Expandable title="Project Variables">
  <ResponseField name="{{ project_name }}" type="text">
    The name of the current project. **Example:** `{{ project_name }}`
  </ResponseField>
  <ResponseField name="{{ project_version }}" type="text">
    The version of the current project. **Example:** `{{ project_version }}`
  </ResponseField>
</Expandable>

---

### **Custom Variables**

<Expandable title="Custom Variables">
  <ResponseField name="{{ custom_variable1 }}" type="text">
    A custom-defined variable. **Example:** `{{ custom_variable1 }}`
  </ResponseField>
  <ResponseField name="{{ custom_flag }}" type="boolean">
    A custom boolean flag. **Example:** `{{ custom_flag }}`
  </ResponseField>
</Expandable>

---

### **Session Variables**

<Expandable title="Session Variables">
  <ResponseField name="{{ session_id }}" type="text">
    The unique identifier for the current session. **Example:** `
    {{ session_id }}`
  </ResponseField>
  <ResponseField name="{{ session_active }}" type="boolean">
    Indicates if the session is currently active. **Example:** `
    {{ session_active }}`
  </ResponseField>
</Expandable>

---

### **Authentication Variables**

<Expandable title="Authentication Variables">
  <ResponseField name="{{ auth_token }}" type="text">

  </ResponseField>
  <ResponseField name="{{ auth_status }}" type="text">
    The authentication status of the user. **Example:** `{{ auth_status }}`
  </ResponseField>

---

### **System Variables**

<Expandable title="System Variables">
