---
title: 'List Campaigns'
api: 'GET /v1/organizations/{org_id}/campaigns'
description: 'Get all campaigns for an organization'
---

## Authorizations

<ParamField header="X-API-Key" type="string" required>
  Authentication header
</ParamField>

## Path

<ParamField path="org_id" type="string" required>
  Organization unique_id
</ParamField>

## Query

<ParamField query="user_id" type="string" required>
  User unique_id
</ParamField>

<ParamField query="is_admin" type="boolean">
  If true, returns all org campaigns. If false, returns only user's campaigns.
</ParamField>

<RequestExample>

```bash cURL
curl -X GET "https://api.teli.ai/v1/organizations/{org_id}/campaigns?user_id={user_id}&is_admin=true" \
  -H "X-API-Key: YOUR_API_KEY"
```

</RequestExample>
