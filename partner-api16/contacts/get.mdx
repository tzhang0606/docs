---
title: 'Get Contacts'
api: 'GET /v1/contacts/lists/{id}/contacts'
description: 'Get contacts from a list'
---

## Authorizations

<ParamField header="X-API-Key" type="string" required>
  Authentication header
</ParamField>

## Path

<ParamField path="id" type="string" required>
  Contact list unique_id
</ParamField>

## Query Parameters

<ParamField query="user_id" type="string">
  User unique_id (for access control)
</ParamField>

<ParamField query="is_admin" type="boolean">
  If true, admin can view any list in the org. If false/missing, user can only view their own lists.
</ParamField>

<RequestExample>

```bash
curl -X GET "https://api.teli.ai/v1/contacts/lists/{list_id}/contacts?user_id={user_id}&is_admin=true" \
  -H "X-API-Key: YOUR_API_KEY"
```

</RequestExample>
