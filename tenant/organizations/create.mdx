---
title: 'Create Organization'
api: 'POST /v1/organizations'
description: 'Create a new organization'
---

Create a new organization to manage your SMS campaigns, users, and contacts.

<Note>
After creating an organization, you'll need to complete 10DLC registration to obtain an SMS phone number before sending campaigns.
</Note>

## Authorizations

<ParamField header="X-API-Key" type="string" required>
  Authentication header containing your API key. Contact your account manager to obtain your key.
</ParamField>

## Body

<ParamField body="name" type="string" required>
  Organization name.
  
  Example: `"My Company LLC"`
</ParamField>

<ParamField body="contact_email" type="string" required>
  Primary contact email address for the organization.
  
  Example: `"admin@mycompany.com"`
</ParamField>

<RequestExample>

```bash cURL
curl -X POST https://api.example.com/v1/organizations \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Company LLC",
    "contact_email": "admin@mycompany.com"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.example.com/v1/organizations', {
  method: 'POST',
  headers: {
    'X-API-Key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'My Company LLC',
    contact_email: 'admin@mycompany.com'
  })
});

const data = await response.json();
console.log(data.organization.unique_id);
```

```python Python
import requests

response = requests.post(
    'https://api.example.com/v1/organizations',
    headers={'X-API-Key': 'YOUR_API_KEY'},
    json={
        'name': 'My Company LLC',
        'contact_email': 'admin@mycompany.com'
    }
)

data = response.json()
print(data['organization']['unique_id'])
```

## Response Fields

<ResponseField name="unique_id" type="string">
  Organization identifier. Use this in all subsequent API calls.
</ResponseField>

<ResponseField name="sms_active" type="boolean">
  Whether organization can send SMS campaigns. `false` until 10DLC registration is approved.
</ResponseField>

<ResponseField name="teli_sms_number" type="string">
  Assigned 10DLC phone number for sending SMS. `null` until approved.
</ResponseField>

<ResponseField name="sms_registration_status" type="string">
  10DLC registration status: `not_started`, `pending_approval`, `approved`, or `rejected`
</ResponseField>

## Next Steps

After creating an organization:

1. **Create a user** - Add team members with `POST /v1/organizations/{org_id}/users`
2. **Register for 10DLC** - Complete SMS registration to get your phone number
3. **Create SMS agents** - Set up conversation templates
4. **Upload contacts** - Import your lead lists
5. **Launch campaigns** - Start sending SMS
