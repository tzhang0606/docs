## Create Organization

<RequestExample>

```bash cURL
curl -X POST https://your-api.modal.run/v1/organizations \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Company LLC",
    "contact_email": "admin@mycompany.com"
  }'
```

```javascript JavaScript
const response = await fetch('https://your-api.modal.run/v1/organizations', {
  method: 'POST',
  headers: {
    'X-API-Key': 'your_api_key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'My Company LLC',
    contact_email: 'admin@mycompany.com'
  })
});
```

```python Python
import requests

response = requests.post(
    'https://your-api.modal.run/v1/organizations',
    headers={'X-API-Key': 'your_api_key'},
    json={
        'name': 'My Company LLC',
        'contact_email': 'admin@mycompany.com'
    }
)
```

</RequestExample>

## 10DLC Registration (Optional)

Before sending SMS campaigns, organizations need a 10DLC phone number.

### Check Registration Status

```bash
GET /v1/organizations/{org_id}
```

Look for `sms_registration_status`:
- `not_started`: Need to register
- `pending_approval`: Waiting for carrier approval (1-2 days)
- `approved`: Ready to send SMS!

### When Approved

Once `sms_active: true`, you'll receive:
- `teli_sms_number`: Your dedicated SMS phone number
- Full campaign sending capabilities

<Note>
10DLC registration endpoints are covered in a separate section. Contact support for registration assistance.
</Note>
