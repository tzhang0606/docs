## Description

Returns all available text-to-speech voices that can be used when creating or updating voice agents. Use the `voice_id` from the response when setting the voice for your agents.

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Your Teli API key
</ParamField>

## Response Fields

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="count" type="integer">
  Total number of available voices
</ResponseField>

<ResponseField name="voices" type="array">
  Array of voice objects
  
  <Expandable title="Voice Object">
    <ResponseField name="voice_id" type="string">
      Unique voice identifier. Use this value when creating/updating agents.
      Example: `"11labs-Adrian"`
    </ResponseField>
    
    <ResponseField name="voice_name" type="string">
      Human-readable voice name.
      Example: `"Adrian"`
    </ResponseField>
    
    <ResponseField name="provider" type="string">
      Voice provider. Always returns `"teli-voice"`.
    </ResponseField>
    
    <ResponseField name="gender" type="string">
      Voice gender: `"male"` or `"female"`
    </ResponseField>
    
    <ResponseField name="accent" type="string">
      Voice accent (e.g., `"American"`, `"British"`, `"Australian"`)
    </ResponseField>
    
    <ResponseField name="age" type="string">
      Voice age category (e.g., `"Young"`, `"Middle-aged"`, `"Old"`)
    </ResponseField>
    
    <ResponseField name="language" type="string">
      Voice language code (e.g., `"en-US"`, `"es-ES"`)
    </ResponseField>
    
    <ResponseField name="preview_url" type="string">
      URL to an audio sample of the voice. Use this to let users preview voices before selecting.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="powered_by" type="string">
  Always returns `"Teli"`
</ResponseField>

## Usage with Agents

Use the `voice_id` when creating or updating a voice agent:

```json
{
  "agent_type": "voice",
  "agent_name": "Sales Agent",
  "voice_id": "11labs-Adrian",
  "starting_message": "Hello! How can I help you today?",
  "prompt": "You are a helpful sales assistant..."
}
```

## Voice Providers

All voices are provided through Teli Voice and include voices from multiple high-quality TTS providers:

| Provider | Voice ID Prefix | Quality |
|----------|-----------------|---------|
| Teli Voice (ElevenLabs) | `11labs-` | Ultra-realistic |
| Teli Voice (OpenAI) | `openai-` | Natural |
| Teli Voice (Deepgram) | `deepgram-` | Fast |
| Teli Voice (Cartesia) | `cartesia-` | Expressive |

## Notes

- Voice availability may vary based on your plan
- Some voices support multiple languages
- Voice IDs are stable and can be stored for agent configuration
- Use gender and accent filters in your UI to help users find the right voice
- The `preview_url` provides an audio sample - embed it in your UI so users can hear the voice before selecting
- Preview audio files are hosted on Teli's CDN for fast, reliable playback
