---
title: 'Delete Knowledge Base Source'
api: 'DELETE /v1/knowledge-bases/{id}/sources/{source_id}'
description: 'Remove a specific source from knowledge base'
---

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Your Teli API key
</ParamField>

---

## Path Parameters

<ParamField path="id" type="string" required>
  Knowledge base ID
</ParamField>

<ParamField path="source_id" type="string" required>
  Source ID to delete (e.g., "kb_source_037703c723bf1310")
</ParamField>

---

## Request Examples

<CodeGroup>

```bash cURL
curl -X DELETE "https://api.teli.ai/v1/knowledge-bases/knowledge_base_7a0682ce5fc02413/sources/kb_source_037703c723bf1310" \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.teli.ai/v1/knowledge-bases/kb_xxx/sources/source_xxx',
  {
    method: 'DELETE',
    headers: { 'X-API-Key': 'YOUR_API_KEY' }
  }
)
```

```python Python
import requests

response = requests.delete(
    'https://api.teli.ai/v1/knowledge-bases/kb_xxx/sources/source_xxx',
    headers={'X-API-Key': 'YOUR_API_KEY'}
)
```

</CodeGroup>

---

## Response

<ResponseExample>

```json 200 - Success
{
  "success": true,
  "message": "Source deleted successfully",
  "total_sources": 1,
  "powered_by": "Teli"
}
```

```json 404 - KB Not Found
{
  "error": "Knowledge base not found",
  "success": false,
  "powered_by": "Teli"
}
```

```json 500 - Delete Failed
{
  "error": "Failed to delete source",
  "success": false,
  "powered_by": "Teli"
}
```

</ResponseExample>

## Get Source ID

**To get source_id values**, use [Get Knowledge Base](/api-reference/knowledge-bases/get):

```json
GET /v1/knowledge-bases/{id}

Response includes:
{
  "knowledge_base": {
    "knowledge_base_sources": [
      {
        "type": "text",
        "source_id": "kb_source_037703c723bf1310",
        "title": "Policy"
      },
      {
        "type": "url",
        "source_id": "kb_source_d54a1329b1464524",
        "url": "https://www.prantanirbarua.dev/"
      }
    ]
  }
}
```

Use the `source_id` to delete specific sources.
